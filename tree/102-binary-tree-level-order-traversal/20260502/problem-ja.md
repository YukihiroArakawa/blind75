# 102. 二分木のレベル順走査

難易度: Medium

## 問題

二分木の `root` が与えられたとき、*そのノードの値をレベル順に走査した結果*を返してください（左から右へ、レベルごとに）。

## 例

**例 1:**

```
入力: root = [3,9,20,null,null,15,7]
出力: [[3],[9,20],[15,7]]
```

**例 2:**

```
入力: root = [1]
出力: [[1]]
```

**例 3:**

```
入力: root = []
出力: []
```

## 制約

- 木のノード数の範囲は `[0, 2000]`
- `-1000 <= Node.val <= 1000`
