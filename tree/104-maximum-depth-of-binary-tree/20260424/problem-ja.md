# 104. 二分木の最大の深さ

難易度: Easy

## 問題

二分木の `root` が与えられたとき、*その最大の深さ*を返してください。

二分木の**最大の深さ**とは、ルートノードから最も遠い葉ノードまでの最長パスに含まれるノードの数です。

## 例

**例 1:**

![tree](https://assets.leetcode.com/uploads/2020/11/26/tmp-tree.jpg)

```
入力: root = [3,9,20,null,null,15,7]
出力: 3
```

**例 2:**

```
入力: root = [1,null,2]
出力: 2
```

## 制約

- 木のノード数は `[0, 10^4]` の範囲内。
- `-100 <= Node.val <= 100`
