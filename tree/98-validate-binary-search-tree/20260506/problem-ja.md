# 98. 二分探索木の検証

難易度: Medium

## 問題

二分木の `root` が与えられます。これが有効な二分探索木（BST）であるかどうかを判定してください。

**有効な BST** は次のように定義されます。

- あるノードの左部分木には、そのノードのキーより **厳密に小さい** キーを持つノードだけが含まれる。
- あるノードの右部分木には、そのノードのキーより **厳密に大きい** キーを持つノードだけが含まれる。
- 左右の部分木もまた二分探索木でなければならない。

## 例

### 例 1

**入力:** `root = [2,1,3]`  
**出力:** `true`

### 例 2

**入力:** `root = [5,1,4,null,null,3,6]`  
**出力:** `false`  
**説明:** 根ノードの値は `5` ですが、その右の子ノードの値は `4` です。

## 制約

- 木に含まれるノード数は `[1, 10^4]` の範囲にある。
- `-2^31 <= Node.val <= 2^31 - 1`
