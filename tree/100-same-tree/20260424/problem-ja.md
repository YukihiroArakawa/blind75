# 100. 同じ木

難易度: Easy

## 問題

2つの二分木のルート `p` と `q` が与えられたとき、それらが同じかどうかを判定する関数を書いてください。

2つの二分木は、**構造的に同一**であり、かつ**各ノードの値が等しい**ときに同じとみなされます。

## 例

**例 1:**

![ex1](https://assets.leetcode.com/uploads/2020/12/20/ex1.jpg)

```
入力: p = [1,2,3], q = [1,2,3]
出力: true
```

**例 2:**

![ex2](https://assets.leetcode.com/uploads/2020/12/20/ex2.jpg)

```
入力: p = [1,2], q = [1,null,2]
出力: false
```

**例 3:**

![ex3](https://assets.leetcode.com/uploads/2020/12/20/ex3.jpg)

```
入力: p = [1,2,1], q = [1,1,2]
出力: false
```

## 制約

- どちらの木のノード数も `[0, 100]` の範囲内。
- `-10^4 <= Node.val <= 10^4`
