# 20. 有効な括弧

難易度: Easy

## 問題

`'('`, `')'`, `'{'`, `'}'`, `'['`, `']'` の文字のみを含む文字列 `s` が与えられる。入力文字列が有効であるかを判定せよ。

入力文字列が有効であるとは、以下を満たすことである:

1. 開き括弧は同じ種類の閉じ括弧で閉じられなければならない。
2. 開き括弧は正しい順序で閉じられなければならない。
3. すべての閉じ括弧には、対応する同じ種類の開き括弧が存在しなければならない。

## 例

**例 1:**

```
入力: s = "()"
出力: true
```

**例 2:**

```
入力: s = "()[]{}"
出力: true
```

**例 3:**

```
入力: s = "(]"
出力: false
```

**例 4:**

```
入力: s = "([])"
出力: true
```

**例 5:**

```
入力: s = "([)]"
出力: false
```

## 制約

- `1 <= s.length <= 10^4`
- `s` は括弧 `'()[]{}'` のみで構成される。
