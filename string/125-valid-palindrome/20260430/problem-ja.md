# 125. 有効な回文

難易度: Easy

## 問題

ある文字列が **回文 (palindrome)** であるとは、すべての大文字を小文字に変換し、英数字以外の文字をすべて取り除いた後に、前から読んでも後ろから読んでも同じになることをいう。英数字とは、アルファベットと数字を指す。

文字列 `s` が与えられたとき、それが **回文** であれば `true` を、そうでなければ `false` を返せ。

## 例

**例 1:**

```
入力: s = "A man, a plan, a canal: Panama"
出力: true
説明: "amanaplanacanalpanama" は回文である。
```

**例 2:**

```
入力: s = "race a car"
出力: false
説明: "raceacar" は回文ではない。
```

**例 3:**

```
入力: s = " "
出力: true
説明: 英数字以外を取り除くと s は空文字列 "" になる。
空文字列は前から読んでも後ろから読んでも同じなので、回文である。
```

## 制約

- `1 <= s.length <= 2 * 10^5`
- `s` は印字可能な ASCII 文字のみで構成される。
