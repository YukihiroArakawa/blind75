# 647. 回文部分文字列

難易度: Medium

## 問題

文字列 `s` が与えられます。`s` に含まれる **回文部分文字列** の個数を返してください。

文字列が前から読んでも後ろから読んでも同じであるとき、その文字列は **回文** です。

**部分文字列** とは、文字列の中の連続した文字列です。

## 例

### 例 1

**入力:** `s = "abc"`  
**出力:** `3`  
**説明:** 回文文字列は `"a"`、`"b"`、`"c"` の 3 つです。

### 例 2

**入力:** `s = "aaa"`  
**出力:** `6`  
**説明:** 回文文字列は `"a"`、`"a"`、`"a"`、`"aa"`、`"aa"`、`"aaa"` の 6 つです。

## 制約

- `1 <= s.length <= 1000`
- `s` は小文字の英字のみから構成される。
