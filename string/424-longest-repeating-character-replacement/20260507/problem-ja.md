# 424. 最長の繰り返し文字置換

難易度: Medium

## 問題

文字列 `s` と整数 `k` が与えられます。文字列中の任意の文字を選び、それを別の任意の大文字英字に変更できます。この操作は高々 `k` 回まで行えます。

上の操作を行ったあとに得られる、「同じ文字だけを含む最長の部分文字列」の長さを返してください。

## 例

### 例 1

**入力:** `s = "ABAB", k = 2`  
**出力:** `4`  
**説明:** 2 つの `'A'` を 2 つの `'B'` に置き換えるか、その逆を行います。

### 例 2

**入力:** `s = "AABABBA", k = 1`  
**出力:** `4`  
**説明:** 真ん中の 1 つの `'A'` を `'B'` に置き換えると `"AABBBBA"` になります。  
部分文字列 `"BBBB"` は、同じ文字が繰り返される最長部分文字列で、その長さは `4` です。  
この答えを得る方法は他にもありえます。

## 制約

- `1 <= s.length <= 10^5`
- `s` は大文字の英字のみから構成される。
- `0 <= k <= s.length`
