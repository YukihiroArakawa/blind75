# 271. 文字列リストのエンコードとデコード

難易度: Medium

## 問題

文字列のリストを 1 つの文字列にエンコードし、その文字列をネットワーク越しに送ったあとで、元の文字列リストへ正しくデコードできるアルゴリズムを設計してください。

`encode` メソッドと `decode` メソッドを実装してください。

`eval`、`serialize`、あるいは言語組み込みの「リストをそのまま文字列化して戻す」ようなシリアライズ機能は使ってはいけません。

アルゴリズムは、次のような任意の文字列を正しく扱える必要があります。

- 空文字列
- 数字を含む文字列
- 区切り文字のような文字を含む文字列

## 例

### 例 1

**入力:** `["lint","code","love","you"]`  
**出力:** `["lint","code","love","you"]`

### 例 2

**入力:** `["we","say",":","yes"]`  
**出力:** `["we","say",":","yes"]`

## 制約

- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`
- `strs[i]` は任意の有効な ASCII 文字を含みうる。
