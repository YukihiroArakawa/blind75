# 49. アナグラムのグループ化

難易度: Medium

## 問題

文字列配列 `strs` が与えられます。アナグラム同士を同じグループにまとめてください。答えは **任意の順序** で返してかまいません。

## 例

### 例 1

**入力:** `strs = ["eat","tea","tan","ate","nat","bat"]`  
**出力:** `[["bat"],["nat","tan"],["ate","eat","tea"]]`

**説明:**

- `strs` の中に `"bat"` に並べ替えられる文字列はありません。
- `"nat"` と `"tan"` は、並べ替えることで互いに同じ文字列にできるのでアナグラムです。
- `"ate"`、`"eat"`、`"tea"` は、並べ替えることで互いに同じ文字列にできるのでアナグラムです。

### 例 2

**入力:** `strs = [""]`  
**出力:** `[[""]]`

### 例 3

**入力:** `strs = ["a"]`  
**出力:** `[["a"]]`

## 制約

- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`
- `strs[i]` は小文字の英字のみから構成される。
