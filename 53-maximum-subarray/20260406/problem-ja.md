# 53. 最大部分配列

難易度: Medium

## 問題

整数配列 `nums` が与えられたとき、合計が最大となる部分配列（連続する要素の並び）を見つけ、その合計を返してください。

## 例

**例 1:**

```
入力: nums = [-2,1,-3,4,-1,2,1,-5,4]
出力: 6
説明: 部分配列 [4,-1,2,1] が最大の合計 6 を持つ。
```

**例 2:**

```
入力: nums = [1]
出力: 1
説明: 部分配列 [1] が最大の合計 1 を持つ。
```

**例 3:**

```
入力: nums = [5,4,-1,7,8]
出力: 23
説明: 部分配列 [5,4,-1,7,8] が最大の合計 23 を持つ。
```

## 制約

- `1 <= nums.length <= 10^5`
- `-10^4 <= nums[i] <= 10^4`

## 発展問題

`O(n)` の解法がわかったら、**分割統治法**を使った別の解法にも挑戦してみてください。こちらの方がより巧妙です。
