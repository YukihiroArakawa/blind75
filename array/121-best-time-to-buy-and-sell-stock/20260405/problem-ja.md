# 121. ベストな株の売買タイミング

難易度: Easy

## 問題

ある株の `i` 日目の価格を示す配列 `prices` が与えられます。

ある日にその株を **1回だけ** 買い、将来の **別の日** にその株を売ることで利益を最大化したいと考えています。

この取引から得られる *最大利益* を返してください。利益を得ることができない場合は、`0` を返してください。

## 例

**例 1:**

```
入力: prices = [7,1,5,3,6,4]
出力: 5
説明: 2 日目 (価格 = 1) に買い、5 日目 (価格 = 6) に売ることで、利益 = 6-1 = 5 となります。
なお、2 日目に買って 1 日目に売ることは、買う前に売ることになるため許可されません。
```

**例 2:**

```
入力: prices = [7,6,4,3,1]
出力: 0
説明: この場合、取引は行われず、最大利益 = 0 となります。
```

## 制約

- `1 <= prices.length <= 10^5`
- `0 <= prices[i] <= 10^4`
