# 152. 最大積部分配列

難易度: Medium

## 問題

整数配列 `nums` が与えられたとき、積が最大となる部分配列（subarray）を見つけ、その積を返してください。

テストケースは答えが **32ビット** 整数に収まるように生成されています。

**注意**: 要素が1つの配列の積は、その要素の値そのものです。

## 例

**例 1:**

```
入力: nums = [2,3,-2,4]
出力: 6
説明: [2,3] の積が最大で 6 になります。
```

**例 2:**

```
入力: nums = [-2,0,-1]
出力: 0
説明: [-2,-1] は部分配列（連続）ではないため、答えは 2 にはなりません。
```

## 制約

- `1 <= nums.length <= 2 * 10^4`
- `-10 <= nums[i] <= 10`
- `nums` の任意の部分配列の積は **32ビット** 整数に収まることが保証されています。
