# 217. 重複の存在判定

難易度: Easy

## 問題

整数配列 `nums` が与えられたとき、配列内に同じ値が**2回以上**出現する場合は `true` を返し、すべての要素が異なる場合は `false` を返してください。

## 例

**例 1:**

**入力:** `nums = [1,2,3,1]`

**出力:** `true`

**説明:**

要素 1 がインデックス 0 と 3 に出現しています。

**例 2:**

**入力:** `nums = [1,2,3,4]`

**出力:** `false`

**説明:**

すべての要素が異なります。

**例 3:**

**入力:** `nums = [1,1,1,3,3,4,3,2,4,2]`

**出力:** `true`

## 制約

- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
