# 238. 自分自身を除いた配列の積

難易度: Medium

## 問題

整数配列 `nums` が与えられたとき、`answer[i]` が `nums[i]` を除く `nums` の全要素の積に等しくなるような配列 `answer` を返してください。

`nums` の任意のプレフィックスまたはサフィックスの積は、**32ビット**整数に収まることが**保証**されています。

`O(n)` の時間計算量で、除算を使わないアルゴリズムを記述してください。

## 例

**例 1:**

```
入力: nums = [1,2,3,4]
出力: [24,12,8,6]
```

**例 2:**

```
入力: nums = [-1,1,0,-3,3]
出力: [0,0,9,0,0]
```

## 制約

- `2 <= nums.length <= 10^5`
- `-30 <= nums[i] <= 30`
- `answer[i]` が**32ビット**整数に収まることが**保証**されるような入力が生成される。

## 発展問題

`O(1)` の追加空間計算量で解くことができますか？（出力配列は空間計算量の分析において追加空間として**カウントしません**。）
