# 15. 3Sum

難易度: Medium

## 問題

整数配列 `nums` が与えられたとき、`i != j`、`i != k`、`j != k` かつ `nums[i] + nums[j] + nums[k] == 0` を満たすすべての三つ組 `[nums[i], nums[j], nums[k]]` を返してください。

解の集合に**重複する三つ組を含めてはいけません**。

## 例

**例 1:**

```
入力: nums = [-1,0,1,2,-1,-4]
出力: [[-1,-1,2],[-1,0,1]]
説明:
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0.
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0.
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0.
重複を除いた三つ組は [-1,0,1] と [-1,-1,2] です。
出力の順序および三つ組内の順序は問いません。
```

**例 2:**

```
入力: nums = [0,1,1]
出力: []
説明: 唯一可能な三つ組の合計は 0 になりません。
```

**例 3:**

```
入力: nums = [0,0,0]
出力: [[0,0,0]]
説明: 唯一可能な三つ組の合計が 0 になります。
```

## 制約

- `3 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`
