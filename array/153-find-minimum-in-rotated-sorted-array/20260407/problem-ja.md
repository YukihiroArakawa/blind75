# 153. 回転ソート済み配列の最小値を見つける

難易度: Medium

## 問題

長さ `n` の昇順ソート済み配列が、`1` 回から `n` 回の間で **回転** されたとします。例えば、配列 `nums = [0,1,2,4,5,6,7]` は次のようになり得ます:

- `4` 回回転された場合: `[4,5,6,7,0,1,2]`
- `7` 回回転された場合: `[0,1,2,4,5,6,7]`

配列 `[a[0], a[1], a[2], ..., a[n-1]]` を 1 回 **回転** すると、`[a[n-1], a[0], a[1], a[2], ..., a[n-2]]` になります。

**ユニーク** な要素からなるソート済み回転配列 `nums` が与えられたとき、*この配列の最小要素* を返してください。

`O(log n) time` で動くアルゴリズムを書く必要があります。

## 例

**例 1:**

```
入力: nums = [3,4,5,1,2]
出力: 1
説明: 元の配列は [1,2,3,4,5] で、3 回回転された。
```

**例 2:**

```
入力: nums = [4,5,6,7,0,1,2]
出力: 0
説明: 元の配列は [0,1,2,4,5,6,7] で、4 回回転された。
```

**例 3:**

```
入力: nums = [11,13,15,17]
出力: 11
説明: 元の配列は [11,13,15,17] で、4 回回転された。
```

## 制約

- `n == nums.length`
- `1 <= n <= 5000`
- `-5000 <= nums[i] <= 5000`
- `nums` のすべての整数は **ユニーク** である。
- `nums` はソート済みで、`1` 回から `n` 回の間で回転されている。
