# 11. 最も水を含むコンテナ

難易度: Medium

## 問題

長さ `n` の整数配列 `height` が与えられます。`i` 番目の線の両端点が `(i, 0)` と `(i, height[i])` となるように、`n` 本の垂直な線が引かれています。

x 軸とともにコンテナを形成する2本の線を選び、最も多くの水を含むコンテナを求めてください。

コンテナが格納できる水の最大量を返してください。

**注意**: コンテナを傾けることはできません。

## 例

**例 1:**

```
入力: height = [1,8,6,2,5,4,8,3,7]
出力: 49
説明: 上記の垂直線は配列 [1,8,6,2,5,4,8,3,7] で表されます。この場合、コンテナが格納できる水の最大面積（青の領域）は 49 です。
```

**例 2:**

```
入力: height = [1,1]
出力: 1
```

## 制約

- `n == height.length`
- `2 <= n <= 10^5`
- `0 <= height[i] <= 10^4`
