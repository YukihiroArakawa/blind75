# 73. 行列のゼロ化

難易度: Medium

## 問題

`m x n` の整数行列 `matrix` が与えられます。ある要素が `0` であれば、その要素を含む行全体と列全体を `0` にしてください。

この操作は **in place**、つまり追加の行列を作らずに行う必要があります。

## 例

**例 1:**

```text
入力: matrix = [[1,1,1],[1,0,1],[1,1,1]]
出力: [[1,0,1],[0,0,0],[1,0,1]]
```

**例 2:**

```text
入力: matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]
出力: [[0,0,0,0],[0,4,5,0],[0,3,1,0]]
```

## 制約

- `m == matrix.length`
- `n == matrix[0].length`
- `1 <= m, n <= 200`
- `-2^31 <= matrix[i][j] <= 2^31 - 1`

## 発展

- `O(mn)` の追加領域を使う素直な解法は、あまり良い方法ではありません。
- `O(m + n)` の追加領域を使う方法はありますが、それでも最善ではありません。
- `O(1)` の追加領域で解けないでしょうか。
