# 191. 1 ビットの数

難易度: Easy

## 問題

正の整数 `n` が与えられます。その 2 進表現に含まれる set bit の数（Hamming weight とも呼ばれます）を返す関数を書いてください。

## 例

**例 1:**

```text
入力: n = 11
出力: 3
説明:
入力の 2 進文字列 `1011` には、set bit が合計 3 個あります。
```

**例 2:**

```text
入力: n = 128
出力: 1
```

**例 3:**

```text
入力: n = 2147483645
出力: 30
説明:
入力の 2 進文字列 `1111111111111111111111111111101` には、set bit が合計 30 個あります。
```

## 制約

- `1 <= n <= 2^31 - 1`

**発展:** この関数が何度も呼ばれる場合、どのように最適化しますか？

## 備考

- set bit とは、2 進表現で `1` になっているビットのことです。
- この問題では、2 進数の中に `1` が何個あるかを数えます。
- ビット演算では `n & (n - 1)` を使うと、`n` の一番右にある `1` を 1 回で消せます。
- その操作を `n` が 0 になるまで繰り返すと、`1` の数だけループが回ります。
