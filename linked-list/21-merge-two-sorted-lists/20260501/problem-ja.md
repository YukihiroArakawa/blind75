# 21. 2 つのソート済み連結リストのマージ

難易度: Easy

## 問題

ソート済み連結リスト 2 つの先頭 `list1` と `list2` が与えられます。

これら 2 つのリストを 1 つの**ソート済み**リストにマージしてください。新しいリストは、元の 2 つのリストのノードをつなぎ合わせて作成する必要があります。

マージ後の連結リストの先頭を返してください。

## 例

**例 1:**

```
入力: list1 = [1,2,4], list2 = [1,3,4]
出力: [1,1,2,3,4,4]
```

**例 2:**

```
入力: list1 = [], list2 = []
出力: []
```

**例 3:**

```
入力: list1 = [], list2 = [0]
出力: [0]
```

## 制約

- 両リストのノード数の範囲は `[0, 50]`
- `-100 <= Node.val <= 100`
- `list1` と `list2` はどちらも**非減少**順にソートされている
