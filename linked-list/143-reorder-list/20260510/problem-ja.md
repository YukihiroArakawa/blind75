# 143. 連結リストの並べ替え

難易度: Medium

## 問題

単方向連結リストの先頭 `head` が与えられます。このリストは次のように表せます。

`L0 -> L1 -> ... -> Ln - 1 -> Ln`

このリストを次の形に並べ替えてください。

`L0 -> Ln -> L1 -> Ln - 1 -> L2 -> Ln - 2 -> ...`

ノードの値を書き換えてはいけません。変更してよいのはノード同士のつながりだけです。

## 例

### 例 1

**入力:** `head = [1,2,3,4]`  
**出力:** `[1,4,2,3]`

### 例 2

**入力:** `head = [1,2,3,4,5]`  
**出力:** `[1,5,2,4,3]`

## 制約

- リスト中のノード数は `[1, 5 * 10^4]` の範囲にある。
- `1 <= Node.val <= 1000`
