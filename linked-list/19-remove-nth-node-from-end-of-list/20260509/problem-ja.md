# 19. リストの末尾から N 番目のノードを削除

難易度: Medium

## 問題

連結リストの `head` が与えられます。末尾から `n^th` 番目のノードを削除し、その連結リストの `head` を返してください。

## 例

### 例 1

**入力:** `head = [1,2,3,4,5], n = 2`  
**出力:** `[1,2,3,5]`

### 例 2

**入力:** `head = [1], n = 1`  
**出力:** `[]`

### 例 3

**入力:** `head = [1,2], n = 1`  
**出力:** `[1]`

## 制約

- リスト中のノード数は `sz`。
- `1 <= sz <= 30`
- `0 <= Node.val <= 100`
- `1 <= n <= sz`
