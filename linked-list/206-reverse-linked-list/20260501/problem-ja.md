# 206. 連結リストの反転

難易度: Easy

## 問題

単方向連結リストの先頭ノード `head` が与えられたとき、リストを反転させ、*反転後のリスト* を返せ。

## 例

**例 1:**

```
入力: head = [1,2,3,4,5]
出力: [5,4,3,2,1]
```

**例 2:**

```
入力: head = [1,2]
出力: [2,1]
```

**例 3:**

```
入力: head = []
出力: []
```

## 制約

- リスト中のノード数は `[0, 5000]` の範囲。
- `-5000 <= Node.val <= 5000`
