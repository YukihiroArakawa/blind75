# 323. 無向グラフの連結成分数

難易度: Medium

## 問題

`n` 個のノードからなるグラフがあります。整数 `n` と配列 `edges` が与えられ、`edges[i] = [ai, bi]` はノード `ai` とノード `bi` の間に辺があることを表します。

このグラフに含まれる **連結成分の個数** を返してください。

## 例

**例 1:**

```text
入力: n = 5, edges = [[0,1],[1,2],[3,4]]
出力: 2
```

ノード `0, 1, 2` が 1 つの連結成分、ノード `3, 4` がもう 1 つの連結成分なので、答えは `2` です。

**例 2:**

```text
入力: n = 5, edges = [[0,1],[1,2],[2,3],[3,4]]
出力: 1
```

すべてのノードが 1 つにつながっているので、連結成分は `1` 個です。

## 制約

- `1 <= n <= 2000`
- `1 <= edges.length <= 5000`
- `edges[i].length == 2`
- `0 <= ai <= bi < n`
- `ai != bi`
- 重複した辺は存在しません。

## 備考

- 連結成分とは、互いに到達可能なノードの最大集合のことです。
- 辺が 1 本もつながっていない単独ノードも、1 つの連結成分として数えます。
