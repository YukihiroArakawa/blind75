# 261. 有効な木グラフ

難易度: Medium

## 問題

`0` から `n - 1` までの番号が付いた `n` 個のノードからなるグラフがあります。整数 `n` と `edges` が与えられ、`edges[i] = [ai, bi]` はノード `ai` とノード `bi` の間に **無向辺** があることを表します。

与えられた `edges` が **有効な木** を構成しているなら `true` を、そうでなければ `false` を返してください。

## 例

**例 1:**

```text
入力: n = 5, edges = [[0,1],[0,2],[0,3],[1,4]]
出力: true
```

このグラフは全ノードが 1 つにつながっていて閉路もないため、木です。

**例 2:**

```text
入力: n = 5, edges = [[0,1],[1,2],[2,3],[1,3],[1,4]]
出力: false
```

このグラフには `1 -> 2 -> 3 -> 1` の閉路があるため、木ではありません。

## 制約

- `1 <= n <= 2000`
- `0 <= edges.length <= 5000`
- `edges[i].length == 2`
- `0 <= ai, bi < n`
- `ai != bi`
- 自己ループや重複した辺は存在しません。

## 備考

- 木であるためには、**閉路がないこと** と **すべてのノードが 1 つに連結していること** の両方が必要です。
- ノード数が `n` の木の辺数は必ず `n - 1` 本です。
